  • 网络依赖
  • 定制化受限

混合部署方案
兼顾灵活性与安全性

方案思路：非敏感业务使用云服务（如公开知识库、通用问答），敏感数据本地化处理（如内部文档、涉密资料）。通过网关层统一调度，实现成本与安全的平衡。

典型架构：前端应用 → API网关 → 敏感数据识别 → 涉密内容走本地LLM，公开内容走云端LLM → 统一响应
